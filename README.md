# projectSubmission
## Language versions that we're using
### Node: LTS (~6.10.0)
#### Node Packages: See [package.json](./package.json)

## Install
### NodeJS
```bash
npm install
```

Copy config.example.js to config.js

Edit config.js with info (You will need to create a Google Oauth Application)

## Run
### NodeJS
```bash
npm start
```
